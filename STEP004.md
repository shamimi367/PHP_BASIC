## 連想配列
配列の要素指定(添字)に整数ではなく文字列を使用します。  
要素に順番ではなく、名前(key)をつけることが出来ます。  
```
<?php
$scores = array(
    "math" => 80,
    "english" => 65,
    "japanese"   => 75,
    "history"  => 66,
    "science"  => 69,
);
var_dump($scores);
```
